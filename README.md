# Weton
Weton adalah aplikasi android sedrhana yang berfungsi untuk menghitung neptu; 

## Features
- Dapat menghitung hari dan pasar dari tanggal lahir.
- Dapat menghitung sio lahir.
- Dapat menghitung kecocokan jodoh dengan beberapa metode.
- Dilengkapi kalender sepanjang masa, untuk mencari secara manual hari dan pasra berdasarkan tanggal lahir.

## Tech
- Android Studio 2023.1.1.28
- JDK 8
